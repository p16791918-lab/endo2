## [갑상샘 기능 조절약물 — 김치대]

- **핵심 키워드**: Thionamide, Levothyroxine, T3/T4 전환, TBG 결합, 방사성 요오드
- **핵심 개념 요약**:
  - 갑상선 호르몬은 대부분 TBG와 결합(비활성형). free form이 활성형이며 T4 free는 전체의 0.03% [PP]
  - T4→T3 전환 억제 약물: propylthiouracil (peroxidase 억제 + T4→T3 전환 억제), methimazole/carbimazole (peroxidase 억제만)
  - TBG 결합 증가 요인 → free T4 감소 → TSH 증가 (2018~2021 빈출 ★)
  - Levothyroxine 용량 증가 요인: antacid·PPI·칼슘제(흡수↓), Rifampin·Carbamazepine(대사↑), Estrogen·임신
  - 131I: 반감기 8일, 갑상선 실질세포 선택 파괴. 임산부 주의 [PP]
- **교수 강조 포인트**:
  - TBG 결합 변화 요인 [2018~2021 빈출 ★★★]
  - propylthiouracil vs methimazole 차이점 (T4→T3 전환 억제 여부) [PP, 2019出]
  - Levothyroxine 흡수 방해 약물 목록 [2022出 ★]
  - L-T3(liothyronine): myxedema coma 등 응급 시 사용 [PP, 2021出]

## [갑상선질환 외과치료 — 이진춘]

- **핵심 키워드**: Graves' disease 수술적응증, 갑상선암 수술범위, 합병증, lobectomy vs total thyroidectomy
- **핵심 개념 요약**:
  - Graves' disease 절대적응증: 임신/임신계획(6~12개월 내), 악성 의심, 빠른 증상 조절 필요 시
  - 상대적응증: large goiter, severe ophthalmopathy, poor compliance, children [2021-2出]
  - 엽절제술(lobectomy): 2cm 미만, 단일 병소, 피막침범 없음, 림프절 전이 없음, 저위험군 [2018, 2020出]
  - 근전절제술/전절제술: 원발암 >4cm, 양측 결절, 갑상선 외 침윤, 원격전이, 45세 이상 고령 [2019~2022出]
  - 합병증: 되돌이후두신경 손상(쉰목소리), 부갑상선 손상(손발저림), **출혈=응급수술의 가장 큰 원인** [2019~2021 빈출]
- **교수 강조 포인트**:
  - Graves' disease 절대/상대 적응증 구분 [PPPPP, 2018·2021-2·2022出]
  - 전절제술 적응증 [PPPPP, 2019·2021-1·2022出]
  - 합병증 항목 전체 [2019~2021 매년 출제]
  - 엽절제술 이점 5가지 / 전절제술 이점 6가지 비교

## [성선기능조절 약물 — 배순식]

- **핵심 키워드**: SERM, GnRH receptor agonist, aromatase inhibitor, PR antagonist, PDE5 inhibitor
- **핵심 개념 요약**:
  - Estrogen 합성: Theca cell(androgen) + Granulosa cell(aromatase로 estrogen) 두 세포 협동 [P]
  - SERM: 조직마다 agonist/antagonist 효과가 다름 — tamoxifen(유방암 치료, 자궁내막엔 agonist) [P]
  - GnRH receptor agonist(leuprolide): 연속 투여 → pulsatile 파괴 → LH/FSH 감소 → 전이성 전립선암·자궁내막증 치료
  - 5α-reductase inhibitor(finasteride): DHT 생성 억제 → 전립선비대증 + 탈모 치료 [出]
  - PDE5 inhibitor(sildenafil 등 ~afil): NO→cGMP 경로 유지 → 발기부전 치료 [P]
- **교수 강조 포인트**:
  - Mifepristone 기전: PR antagonist → prostaglandin↑ → 자궁 수축 → 치료적 유산 [P]
  - tamoxifen vs fulvestrant: tamoxifen 내성 유방암에 2차로 fulvestrant [P]
  - GnRH agonist 역설적 억제 기전 (pulsatile 파괴)
  - 전립선암: GnRH agonist, AR antagonist / 전립선비대증: 5α-reductase inhibitor

## [성호르몬 작용, 평가 — 박현준]

- **핵심 키워드**: Testosterone 일중변동, SHBG, 5α-reductase/DHT, HPG axis, Hypogonadism 분류
- **핵심 개념 요약**:
  - Testosterone: 95%가 고환(Leydig cell)에서 생성 [出]. 나머지는 부신·말초
  - 혈중 T: SHBG 결합 60% + albumin 결합 38% = free T는 2%만 활성형 [出]
  - DHT 경로: T → 5α-reductase → DHT (전립선, 탈모) / T → aromatase → E2 (지방세포) [出]
  - 일중변동: 아침에 최고 → **오전에 측정하는 것이 원칙** [出]
  - Hypogonadism: hypergonadotropic(고환 문제, LH/FSH↑) vs hypogonadotropic(뇌하수체/시상하부 문제, LH/FSH↓)
- **교수 강조 포인트**:
  - Circadian variation — 오전 측정 원칙 [★出]
  - T 대사 3가지 경로 (DHT, E2, seminiferous tubules) [出]
  - 동적 검사: hCG 자극(고환 이상 확인), clomiphene 자극(뇌하수체 이상), GnRH 자극(시상하부/뇌하수체 구별) [出]
  - 비만 → T 감소 역의 상관관계 [出]

## [남성 갱년기 — 김현우]

- **핵심 키워드**: TDS/LOH, 진단기준 300ng/dl, TRT 금기, PSA/DRE 모니터링, CV risk
- **핵심 개념 요약**:
  - 정의: **증상 + 낮은 테스토스테론** 두 가지 모두 있어야 진단 [★]
  - 진단 기준: 오전 측정, total T <300ng/dl 치료 가능, <230ng/dl 적극 치료 [★, 2022出]
  - TRT 금기: **전립선암, 남성 유방암**(절대), 심한 전립선비대증·red blood cell 과다증(상대)
  - CV risk: TRT가 CV event를 증가시킨다는 명확한 증거 없음. 단, CV event 3~6개월 이내라면 치료 보류 [★]
  - 치료 시작: short acting(4주 근주·gel) 먼저 → 부작용 확인 후 long acting(12주) 전환 [★]
- **교수 강조 포인트**:
  - 진단 수치 300(또는 350)/230 ng/dl 암기 [★, 2022出]
  - TRT 원칙: 반드시 **증상 + 수치 감소** 동시 확인 후 시작 [★, 2022出]
  - 치료 전 검사: DRE + PSA (치료 전, 3~6개월, 12개월) [★]
  - 치료 평가: 3~6개월(libido·근기능), 2년(골밀도) [★]

## [갑상선 영상의학 — 김학진]

- **핵심 키워드**: US vs CT, cystic/solid 감별, benign/malignant 초음파소견, microcalcification, TUS 분류
- **핵심 개념 요약**:
  - 갑상선 병변 감별: **US가 CT보다 우월** [P, 出出]. CT는 extent 확인·원격전이 확인용
  - Cystic mass: anechoic + well-defined + posterior enhancement → 양성 多 [PPPP]
  - Solid mass: hypo- or isoechoic, posterior enhancement 없음 [PPPP]
  - 양성 시사: well-defined halo, comet-tail artifact(colloid cyst), honeycomb appearance, macrocalcification [P]
  - 악성 시사: **microcalcification**(특이도 94%) + **irregular margin** → 진단 정확도 90.5% [PP, 出出]
- **교수 강조 포인트**:
  - TUS 1(양성): cystic nodule + peripheral continuous halo [P]
  - TUS 5(강한 악성): punctate microcalcification + ill-defined margin + 림프절 전이 [P]
  - Malignant 특징 4대 소견: hypoechoic solid, microcalcification, irregular/lobulated margin, capsule involvement [2021出]
  - Macrocalcification/rim calcification → 양성, microcalcification(posterior shadowing 없음) → 악성 [P]
  - Hashimoto thyroiditis US: diffuse lobulated + decreased echo + micronodulation
