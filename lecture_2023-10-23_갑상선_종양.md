# 갑상선 종양 종합본 — 2023-10-23 (김보현 교수님)

*강의록 + 정리족 + 출족(2010-2022) 통합*

---

## 1. 갑상선 결절 접근 알고리즘 ★★★★★

```
갑상선 결절 발견
       ↓
혈청 TSH 측정 (첫 번째 검사 PPP)
  ├─ TSH 낮음 → Radionuclide scan → Hot nodule → 99% 양성 (FNA 불필요)
  └─ TSH 정상/높음 → 초음파 평가 → K-TIRADS → FNA 결정
```

**Hot nodule (TSH 낮음):** 거의 항상 양성 → FNA 시행 안 함 (出 2013, 2015기말, 2019)

## 2. 악성 위험인자 ★★★★★ (매년 출제)

| 병력 위험인자 | 초음파 악성 소견 |
|---|---|
| **남자** | **Solid hypoechoic** |
| **20세 이하 or 65세 이상** | **Microcalcification** |
| **목소리 변화 / 성대마비** | **Taller than wide** |
| **경부 LN 촉지** | **Irregular margin** |

| **방사선 조사력 (두경부)** | |
| **Familial polyposis ★** | |
| **MEN2/FMTC / RET 변이** | |
| **혈청 calcitonin ≥100 pg/mL** | |
| **18F-FDG PET 양성** | |

> **Familial polyposis(가족성 선종성 용종증)**는 매우 자주 출제됨 (2017, 2019, 2021-1)
> 흡연력, 여성, 37세, 소량 음주 등은 위험인자 **아님**

## 3. K-TIRADS / FNA 기준 ★★★★

| 등급 | 특징 | 악성률 | FNA 기준 |
|---|---|---|---|
| **5 (High)** | Solid hypoechoic + micro석회화/taller than wide/irregular margin 중 1개 이상 | >60% | **≥1.0cm** |
| **4 (Intermediate)** | Hypoechoic 결절 suspicious features 없음, or 부분낭성+suspicious | 10-40% | ≥1.0~1.5cm |
| **3 (Low)** | 부분낭성 or iso/hyperechoic + suspicious features 없음 | 3-10% | ≥2.0cm |
| **2 (Benign)** | Spongiform / comet-tail artifact / pure cyst | 매우 낮음 | 루틴 FNA 불필요 |

**양성 소견:** Comet-tail sign ★, hot nodule, peripheral continuous halo

## 4. FNA (Bethesda) 결과별 처치 ★★★

| Bethesda | 처치 |
|---|---|
| 1 (Nondiagnostic) | US-guided FNA 반복 → 그래도 안되면 수술 |
| 2 (Benign) | 정기 추적 |
| 3 (AUS/FLUS) | Core-needle biopsy → 불가 시 BRAF mutation 검사 + FNA 반복 → **BRAF (+)이면 99% 암 → 수술** |
| 4 (Follicular neoplasm) | Diagnostic lobectomy → capsule/vascular invasion 확인 → FTC vs 선종 |
| 5 (Suspicious) | 수술 |
| 6 (Malignant) | 수술 |

> **FTC는 FNA로 진단 불가** — 조직에서 capsular/vascular invasion 확인해야만 악성 확정 (PPP 出)

## 5. 갑상선암 종류별 핵심 ★★★★★

### PTC (유두암) — 우리나라 95%, 전체 80-85%

- Orphan-Annie nuclei, psammoma bodies, papillary structures
- **BRAF V600E** 가장 흔한 유전자 변이 ★★★★★ 出出出
- 림프관 전이 多, 예후 매우 좋음 (5년 생존율 99%)
- Tumor marker: **Thyroglobulin (Tg)** ★★★★★
- NIFTP: noninvasive follicular thyroid neoplasm with papillary-like nuclear features → 더 이상 악성 아님

### FTC (여포암)

- 요오드 결핍 지역 多, 단 5%
- **FNA로 진단 불가** → Lobectomy 후 capsular/vascular invasion 확인
- Minimally invasive → Lobectomy로 충분
- Hematogenous 전이 (bone, lung) — 출 2017 고관절 골절 케이스
- Tumor marker: **Thyroglobulin (Tg)**

### MTC (수질암)

- 5%, **Calcitonin, CEA** 표지자 ★★★★★ 出出出
- **RET point mutation** ★★★★★ — 모든 MTC에서 검사 필수
  - MEN 2A: Cys634Arg / MEN 2B: Met918Thr (가장 공격적) / FMTC: Cys618Arg
- 수술이 **유일한 완치** 치료 ★★★ 出
- RAI 흡수 안 됨
- Sporadic or 가족성 (MEN 2A > MEN 2B > familial MTC 순 공격성)

### ATC (역형성암)

- 진단 후 대부분 6개월 내 사망
- BRAF V600E 25% → **Vemurafenib** (NEJM 2013)
- RAI 흡수 안 됨, 화학요법 거의 효과 없음

### Thyroid Lymphoma

- **Hashimoto's thyroiditis 배경** ★★★ 出
- 급속 커지는 갑상선 종괴
- DLBCL (diffuse large-cell lymphoma) 가장 흔함
- **외부 방사선 치료** 민감 ★★★ 出
- **수술은 초기 치료 금기** (spread 위험) ★★★ 出

## 6. Toxic Adenoma / Toxic MNG ★★★ 出出出 (2014, 2019, 2020, 2021-2)

**전형적 케이스:** 체중감소, 가슴떨림, 더위 못 참음 + TSH↓ + FT4↑ + **TSH receptor Ab 정상(음성)** + Scan에서 단일 hot nodule

- 원인: TSH-R 또는 Gsα **activating mutation**
- 치료: **RAI (방사성요오드) 1차** ★★★★ 出
- Lobectomy도 효과적
- 항갑상선제는 장기치료 효과 없음 (remission 안 됨)

---

## 7. DTC (분화갑상선암) 치료 ★★★

**수술**

- T1a (≤1cm intrathyroidal) → active surveillance 가능
- T1b~T2 (1-4cm) → Lobectomy or near-total thyroidectomy
- T3↑ or 전이 → Near-total thyroidectomy

**TSH 억제:** 전이 확인 시 TSH **<0.1 mIU/L**

**RAI 조건**

- 저요오드 식이 1-2주
- TSH ≥25 mIU/L (T4 중단 or rhTSH 투여)
- **NIS** 발현이 uptake 결정 ★★★★ 出出出 (2018, 2021-1, 2022)

**Follow-up**

- **Thyroglobulin** (anti-Tg Ab 없어야 유효) → 재발/전이 표지자
- **Neck US** — 6개월 후 (PTC 재발의 대부분은 cervical LN)
- Tg↑ & scan(−) → **PET-CT**

## 8. 유전자 정리 ★★★★★

| 암종 | 주요 유전자 |
|---|---|
| PTC | **BRAF V600E** (가장 흔함), RET/PTC 재배열 (20-40%), RAS |
| FTC | **RAS**, PAX8-PPARγ 재배열 |
| MTC | **RET** point mutation |
| ATC | BRAF, TERT promoter, **CTNNB1** (β-catenin), P53 |

> 동일 종양에서 RET, BRAF, RAS 동시 변이는 드묾 → **MAPK cascade 활성화**가 공통 경로

## 10. 암기 포인트 TOP 10

1. 첫 검사 = **혈청 TSH** → 낮으면 scan, 정상/높으면 US
2. **Hot nodule = 99% 양성** → FNA 불필요
3. 악성 위험인자: **Familial polyposis** ★ 반복 출제
4. **FTC = FNA로 진단 불가** (capsular/vascular invasion 조직 확인)
5. **BRAF V600E** = PTC 가장 흔한 유전자, AUS/FLUS에서 양성이면 99% 암
6. **NIS** = RAI 치료 가능 원리, TSH 자극 시 uptake ↑
7. DTC 표지자 = **Tg** (+ anti-Tg Ab 같이 검사) / MTC = **calcitonin + CEA**
8. **Toxic adenoma** 치료 1차 = RAI (항갑상선제 장기 remission 안 됨)
9. **MTC** = RET 변이 필수 검사, 수술 유일 완치, RAI 효과 없음
10. **Thyroid Lymphoma** = Hashimoto 배경, 방사선 치료, 수술 초기 금기
